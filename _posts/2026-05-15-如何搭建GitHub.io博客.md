# 如何搭建 GitHub.io 个人博客

## 为什么选择 GitHub Pages？

GitHub Pages 是 GitHub 提供的免费静态网站托管服务，它有以下优点：

- 完全免费，无需服务器费用
- 支持 Markdown 写作，简单易用
- 集成 Git 版本控制，方便管理
- 支持自定义域名和 HTTPS
- 加载速度快，稳定性高

## 准备工作

在开始之前，你需要：

1. 注册一个 GitHub 账号
2. 了解基本的 Markdown 语法
3. （可选）安装 Git 客户端和文本编辑器
